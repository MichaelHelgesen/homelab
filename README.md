# Homelab

Min struktur på hjemmeserver basert på Proxmox og Docker.

## Status
- Startet: 5. februar 2026

Jeg bygger dette stein for stein og vil forsøke å dokumentere utviklingen på [min nettside](https://www.michaelhelgesen.no).

## Mål
Selvhoste følgende tjenester:
- Jellyfin (media)
- Navidrome (musikk)
- Immich (bilder)
- Nextcloud (filer)
- med mer

## Hardware
- Mac Pro 6,1 (2013 (søppelbøtten))
- Sekskjerners Intel Xeon E5
- 64GB RAM
- Proxmox
